# Lex RedM Portfolio

Premium cinematic RedM developer portfolio built with Next.js + Tailwind + Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy To Vercel

1. Push project to GitHub
2. Import repository into Vercel
3. Deploy instantly

## Features

- Cinematic RedM-inspired UI
- Framer Motion animations
- Mobile responsive
- Client reviews
- Contact form
- Discord integration
- Optimization showcase
- SaaS style presentation
